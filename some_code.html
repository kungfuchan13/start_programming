Okay. Now that you have the latest version of Python installed, we can talk about how you can use it. First off, you should look for something in your python folder called IDLE and open it. Use the search function on your computer if you don't know where it is.

This should open up a text editor-like thing that looks kinda like this:


You can type some stuff there and see what happens. Particularly interesting are numbers and things between quotation marks. Those things do not make IDLE spit out a threatening block of red text, which is good. You can also do things like add and subtract and multiply and divide using the +, -, *, and / operators.

Okay, so you probably want to do more with Python than just use IDLE as a calculator. You can create a New File using the Ctrl+N or Cmd+N command (or from the File menu), from where you can actually start writing a standalone program.
